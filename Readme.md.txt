Bài tập sử dụng git:
B1: Tạo reposotory trên github
B2: Tạo thư mục chứa dự án trên máy tính đặt tên thư mục là GitProject
B3: Clone dự án từ github về thư mục trên máy
B4: Tạo File readme.md đưa nội dung vào
B5: Đẩy file readme.md từ thư mục trên máy tính lên Github
B6: Tạo thư mục source, chứa file index.htlm, news.html. Đưa nội dung bất kỳ vào 2 file.
B7: Đẩy thư mục source chứa 2 file vừa tọa lên Github
